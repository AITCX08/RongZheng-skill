# 戎震.skill

> 基于 B 站 UP 主 **戎震** 公开内容蒸馏的 Claude Code Skill。让 AI 助手以戎震的思维框架与表达 DNA 回应你的问题。

对齐 [女娲.skill](https://github.com/alchaincyf/nuwa-skill) / [张雪峰.skill](https://github.com/alchaincyf/zhangxuefeng-skill) 的格式与使用方式。

---

## 这个 Skill 是什么

`SKILL.md` 是一份「思维操作系统」 —— 它**不是**戎震本人，而是从他公开的 100 个 B 站视频 + 已出版书籍《戎震避坑》72 个章节（共 **172 份素材**）中蒸馏出的：

- 标志金句、核心理念、身份卡
- 心智模型（带 ≥3 条原始证据 + 应用 + 局限）
- 决策启发式（带场景 + 案例）
- 7 维度表达 DNA（句式 / 词汇 / 节奏 / 幽默 / 确定性 / 引用习惯 / 辩论策略）
- 价值观三层（追求 / 拒绝 / 内在张力）
- 时间线、智识谱系、典型问答
- 知识边界与诚实声明

加载后，Claude / Claude Code 会在你触发时**以戎震的视角**分析问题——但请记住，这只是基于公开言论的认知工具，不能替代本人的真实判断。

## 安装

### Claude Code 用户（推荐）

```bash
# 1. clone 到 Claude Code 的 skills 目录
git clone https://github.com/CJWang-bilibili/RongZheng-skill.git \
  ~/.claude/skills/rongzheng

# 2. 重启 Claude Code 即可识别
```

### Claude Desktop / 其他客户端

把 `SKILL.md` 内容粘贴到你的对话上下文 / system prompt / Project 资料里即可。

---

## 使用

加载后，触发短语示例：

```
帮我用戎震的视角分析一下：30 岁要不要回老家进体制？
切换到戎震模式，看看这个亲密关系问题
戎震会怎么看「裸辞做自媒体」这件事？
```

Skill 内置了 Agentic Protocol，会自动以**第一人称（"我"）**回应、用戎震常用的句式与比喻、引用具体心智模型来推理。
